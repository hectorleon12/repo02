# Sintaxis Básica de Markdown

## Títulos
Usa `#` para encabezados:

```markdown
# Título h1
## Título h2
Negrita y Cursiva
Negrita: **texto** o __texto__

Cursiva: *texto* o _texto_

Negrita y Cursiva: ***texto*** o ___texto___

Listas
Desordenada: -, *, o +

Ordenada: 1., 2., etc.

markdown
Copiar
- Item 1
- Item 2
1. Primer
2. Segundo
Enlaces
Para enlaces:

markdown
Copiar
[Texto del enlace](http://url.com)
Imágenes
Insertar imágenes:

markdown
Copiar
![Texto alternativo](http://url.com/imagen.jpg)
Citas
Usa > para citas:

markdown
Copiar
> "Texto citado"
Código
En línea: `código`

Bloque: ```código```
